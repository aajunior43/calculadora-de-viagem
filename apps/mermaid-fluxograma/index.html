▶ Fluxograma

graph TD A[Início] --> B{Decisão} B -->|Sim| C[Ação] B -->|Não| D[Alternativa] C --> E[Fim] D --> E

Use TD (top-down), LR (left-right), RL, BT para direção.

▶ Sequência

sequenceDiagram participant A as Alice participant B as Bob A->>B: Mensagem B-->>A: Resposta A-xB: Mensagem async

Use ->> para seta sólida, -->> para tracejada, -x para assíncrona.

▶ Classes

classDiagram class Animal { +String name +int idade +falar() } Animal <|-- Cachorro

Use <|-- herança, *-- composição, o-- agregação.

▶ Estados

stateDiagram-v2 [*] --> Ativo Ativo --> Inativo Inativo --> Ativo Ativo --> [*]

Use [*] para estado inicial/final, state para sub-estados.

▶ Entidade-Relacionamento

erDiagram USUARIO ||--o{ PEDIDO : faz PEDIDO ||--|{ ITEM : contem USUARIO }|--|{ ENDERECO : possui

Use ||--o{ (um-para-muitos), ||--|| (um-para-um).

▶ Gantt

gantt title Projeto dateFormat YYYY-MM-DD section Fase 1 Tarefa A :a1, 2024-01-01, 30d Tarefa B :after a1, 20d

Use dateFormat para formatar datas e section para agrupar.

▶ Pizza

pie title Vendas "Desktop" : 45 "Mobile" : 35 "Tablet" : 20

Use pie title para adicionar um título ao gráfico.

▶ Jornada

journey title Jornada do Cliente section Pesquisa Visita site: 5: Cliente Busca produto: 3: Cliente section Compra Adiciona carrinho: 4: Cliente Paga: 5: Cliente, Loja

Use números de 1-5 para satisfamento do usuário.

▶ Git Graph

gitGraph commit commit branch develop checkout develop commit checkout main merge develop

Use commit, branch, checkout e merge.
